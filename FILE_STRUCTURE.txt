QUICK FILE REFERENCES
=====================

To add a new screen:
  1. Create: src/screens/FeatureName/FeatureScreen.tsx
  2. Export: src/screens/FeatureName/index.ts
  3. Add to: src/navigation/RootNavigator.tsx
  4. Use colors and spacing from constants

To create UI component:
  1. Create: src/components/ui/ComponentName.tsx
  2. Export from: src/components/ui/index.ts
  3. Import and use in screens

To add new color/spacing:
  1. Edit: src/constants/colors.ts or spacing.ts
  2. Use: import { colors, spacing } from '@/src/constants'

To add global state:
  1. Edit: src/context/AppContext.tsx
  2. Add property to AppContextType interface
  3. Use: const { property } = useAppContext()
